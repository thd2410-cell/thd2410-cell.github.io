# song irum Portfolio

GitHub Pages로 배포하기 위한 개인 포트폴리오 첫 버전입니다.

## 구성

- `index.html` — 포트폴리오 본문
- `styles.css` — 반응형 디자인 스타일

## GitHub Pages 배포 방법

1. GitHub에서 `thd2410-cell.github.io` 이름의 public repository를 만듭니다.
2. 이 폴더의 `index.html`, `styles.css`, `README.md`를 repository 루트에 올립니다.
3. `Settings → Pages`에서 branch를 `main`, folder를 `/root`로 설정합니다.
4. 잠시 뒤 `https://thd2410-cell.github.io` 에서 확인합니다.
